Типы заведений (type):
	1 - Отель
	2 - Кафе
	3 - Ресторан
	4 - Продуктовый
	5 - Фирменный магазин (Опционально для города)

Имя (name):
	Неопционально, может быть на любом языке.

Рейтинг (rate):
	1 - Низший рейтинг заведения
	2 - Плохое заведение
	3 - Более менее
	4 - Неплохое заведение
	5 - Высший рейтинг заведения

Высший отзыв (max_rate):
	может быть от 2 до 5, если  макс. рейтинг 1, то отсутствует.
	Опционально для заведений.

Меньший отзыв (min_rate):
	Может быть от 1 до 4, если мин. рейтинг 5, то отсутствует.
	Опционально для заведений.
